Bank Loan Analysis using SQL
📊 Project Overview

This project analyzes bank loan data to evaluate loan performance, customer behavior, and risk factors. The analysis is performed using SQL and focuses on key financial and business metrics.

📁 Dataset

The dataset contains information about:

Loan Applications

Loan Amount

Total Payment Received

Interest Rate

Debt-to-Income Ratio (DTI)

Loan Status (Fully Paid, Current, Charged Off)

Customer details (State, Employment Length, Home Ownership, Purpose)

🎯 Key Objectives

Analyze total loan applications over time

Evaluate total funded and received amounts

Identify good vs bad loans

Understand loan distribution across different segments

Detect trends and patterns in loan performance

🛠 Tools & Technologies

SQL (SQL Server)

Excel (for basic visualization)

📈 Key Insights

Majority of loans are Fully Paid, indicating a healthy loan portfolio

Charged Off loans represent higher risk but lower overall contribution

Loan applications show variation across months, indicating seasonal trends

Certain states contribute significantly to total loan volume

Customers with stable employment tend to have better loan performance
